Detailed Description

Менеджер пользователей с безопасным хранением паролей.

Обеспечивает регистрацию и аутентификацию пользователей. Пароли никогда не хранятся в открытом виде. Вместо этого:

  • При регистрации генерируется случайная соль длиной 16 символов
  • Пароль комбинируется с солью и хешируется алгоритмом SHA-256
  • В файл users.dat сохраняется только username, salt и passwordHash

При аутентификации:

  • Находится сохраненная соль пользователя
  • Введенный пароль хешируется с той же солью
  • Полученный хеш сравнивается с сохраненным

Такой подход обеспечивает защиту от:

  • Чтения паролей в открытом виде из файла
  • Атак по радужным таблицам (за счет уникальной соли)
  • Восстановления пароля по хешу (SHA-256 является односторонней функцией)

Все публичные методы потокобезопасны (защищены std::mutex).
